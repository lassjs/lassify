## Install

[npm][]:

```sh
npm install lass-cli
```

[yarn][]:

```sh
yarn add lass-cli
```


## Usage

```js
const LassCli = require('lass-cli');

const lassCli = new LassCli();

console.log(lassCli.renderName());
// script
```
